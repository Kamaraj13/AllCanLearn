## Add Your Background Images Here!

Simply drop new images into this folder and run the update script:

```bash
# After adding new images:
python update_slideshow.py
```

Or the slideshow will automatically use all images here - just refresh!

### Current Images
- 20 background images for slideshow
- Logo files (logo.jpg, logo.png)

### Naming Convention
- `bg-*.jpg` - Background images
- Keep filenames simple and descriptive
- No spaces in filenames (use dashes or underscores)

### Supported Formats
- JPG, JPEG, PNG, GIF, WEBP

**Note**: Files with "logo", "icon", or "favicon" in the name are excluded from slideshow
